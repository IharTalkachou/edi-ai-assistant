# EDI ИИ-помощник

Интеллектуальный помощник для технической поддержки и анализа логов системы электронного документооборота.

## Возможности
1. **Диагностика ошибок (RAG):** Анализирует текст ошибки, ищет решение в базе знаний и генерирует понятный ответ клиенту.
2. **Аналитика:** Парсит логи и выявляет системные проблемы с помощью искусственного интеллекта.
3. **Чат-бот:** Интерфейс для общения с документацией.

## Технологический стек
- **Python 3.10**
- **Streamlit** (Web UI)
- **Ollama** (Local LLM Llama 3.1)
- **ChromaDB** (Векторная база знаний)
- **Pandas** (Аналитика данных)

## Установка и запуск

### Предварительные требования
1. Установите [Ollama](https://ollama.com).
2. Скачайте модель: `ollama run llama3.1`

### Запуск локально (для разработки)
```bash
# 1. Клонируйте репозиторий
git clone https://github.com/IharTalkachou/edi-ai-assistant.git

# 2. Создайте виртуальное окружение
python -m venv .venv
source .venv/bin/activate  # или .venv\Scripts\activate на Windows

# 3. Установите зависимости
pip install -r requirements.txt

# 4. Запустите приложение
streamlit run app.py
